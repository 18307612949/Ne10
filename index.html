Project Ne10 has been created to provide a set of common, useful functions that are heavily optimized for the ARM architecture, providing consistent, well-tested behavior that can be easily incorporated into applications. C interfaces to the functions are provided for both assembler and NEON™ implementations, supporting both static and dynamic linking. The library is structured into modules such that functionality that is not required can be ignored.
