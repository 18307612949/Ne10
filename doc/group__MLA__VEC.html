Performs a multiply and accumulate operation using the corresponding elements in acc, src1, and src2.

This function point could be pointed to one of ne10_mla_float_c, ne10_mla_float_neon and ne10_mla_float_asm.

[in]accThese elemtns are added to the result of the multiplication operation
[in]src1The first array to use as the input array
[in]src2The second array to use as the input array
[in]countThe number of items in the two input arrays
